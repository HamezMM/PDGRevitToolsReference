Sheets & Views 6 commands
Manage Sheets
Create, delete, rename, and configure sheets in the active project. Set title blocks, revision clouds, and edit title block parameters from a single dialog.
SheetTools Sheets & Views
Set View Phase
Sets the phase on all eligible views placed on selected sheets. Phase filters are not modified — only the view phase property is updated.
SheetTools Sheets & Views
Create RCP
Creates reflected ceiling plan views from selected floor plans. Matches the crop region exactly (including custom shapes), applies a view template, and places room & ceiling tags automatically.
RCPTools Sheets & Views
Create Existing & Demo Plan
Duplicates selected sheets and all floor plan views on them, applies the CD22 view template, and sets each view's phase to the preceding phase. Produces ready-to-use Existing & Demo plan sheets from your current construction documents.
ExistingDemoTools Sheets & Views
Set View Titles
Automatically repositions viewport title labels across all sheets in the project. Ensures consistent title placement without manual adjustment per sheet.
DraftingTools Sheets & Views
Create Working Drawings
Creates working floor plan views for selected levels in the project. Streamlines the initial drafting setup by generating plan views in bulk from a level selection dialog.
DraftingTools Sheets & Views
Elevation Tools 6 commands
Create Room Elevations
Creates a 4-view elevation set (N/S/E/W) for each selected room. Places elevation markers around the room boundary, names views from the room name and number, and applies a project north orientation.
ElevationTools Elevation Tools
Selection Elevations
Creates N/S/E/W elevation views around the combined bounding box of user-picked elements. Set the marker offset distance (2'–5') from a dialog; applies the CD32 Interior Elevation template automatically.
ElevationTools Elevation Tools
Rotate Crop Region
Rotates the crop region of the active plan view by a user-specified angle. Works with both rectangular and custom-shaped crop regions, rotating around the crop centroid.
ElevationTools Elevation Tools
Reference Plan Creator
Creates reference floor plan callouts linked to selected elevation views. Connects each reference plan back to its source elevation, enabling fast two-way navigation during production drawing.
ElevationTools Elevation Tools
Create Elevation Sheets
Scans all elevation views in the model, groups them by category and room, and packs them onto sheets using a row-fill layout algorithm. Keeps each room's elevation set together on the same sheet.
ElevationSheetTools Elevation Tools
Set Level Head Extents
Trims level line extents to the crop region boundary for every elevation using the CD32 Interior Elevation view template. Extends 1/4" past the boundary wherever a level head bubble is visible.
ElevationTools Elevation Tools
Assembly Tools 5 commands
All Assembly Builder
Runs all four assembly builders (walls, floors, ceilings, roofs) in a single pass across every CD23 sheet. Reports a combined summary of drafting views created and any replaced viewports.
WallAssemblyTools Assembly Tools
Wall Assembly Builder
For every sheet using the CD23 view template, finds all tagged wall types, creates a drafting view containing their matched detail groups, and places it on the sheet. Detail group names must start with the wall type's Type Mark followed by " – ".
WallAssemblyTools Assembly Tools
Floor Assembly Builder
For every CD23 sheet, finds all tagged floor types, creates a drafting view with their matched detail groups, and places it on the sheet. Detail group names must start with the floor type's Type Mark followed by " – ".
FloorAssemblyTools Assembly Tools
Ceiling Assembly Builder
For every CD23 sheet, finds all tagged ceiling types, creates a drafting view with their matched detail groups, and places it on the sheet. Detail group names must start with the ceiling type's Type Mark followed by " – ".
CeilingAssemblyTools Assembly Tools
Roof Assembly Builder
For every CD23 sheet, finds all tagged roof types, creates a drafting view with their matched detail groups, and places it on the sheet. Detail group names must start with the roof type's Type Mark followed by " – ".
RoofAssemblyTools Assembly Tools
Schedules & Doors 4 commands
Schedule Editor
Edit Revit schedule data in a spreadsheet-style dialog. Supports direct cell editing, Excel import and export, and bulk data management — all from within Revit.
DocumentationTools Schedules & Doors
Assembly View Builder
Scans views on selected sheets for walls, floors, ceilings, and roofs whose type mark matches a detail group name. Creates a drafting view per sheet and places the matched detail groups automatically.
DocumentationTools Schedules & Doors
Door Host Wall
Sets the Door's Host Wall parameter on every door instance to the host wall's type mark. Populates this parameter in bulk across the entire model automatically.
DocumentationTools Schedules & Doors
Door Schedule
Fills door schedule parameters — Door's Host Wall, Door FRR, and Frame FRR — by reading the host wall type mark and fire rating, then mapping via the PDG fire rating table.
DocumentationTools Schedules & Doors
Parameters 2 commands
Shared Parameter Creator
Creates shared parameters and binds them to specified Revit categories in the active project. Simplifies shared parameter management without manual file editing.
DocumentationTools Parameters
Family Parameter Converter
Converts family parameters between instance and type within the Family Editor. Avoids manual recreation — flip parameter scope directly from the addin dialog.
DocumentationTools Parameters
Analysis 3 commands
Report Building Area
Updates area and level parameters on all Building Area filled regions in the model. Keeps area schedule data consistent with the current design state.
DraftingTools Analysis
Building Code Analysis

DocumentationTools Analysis
Land Use Analysis

Model Setup 5 commands
Grid Builder
Define and place a full rectilinear structural grid at the model origin. Specify column spacing, bay counts, and grid naming from a single dialog — no manual grid line placement needed.
BaseBuildingTools Model Setup
Find & Replace
Find and replace text in all text notes across the active document. Batch-updates drafting text without needing to locate and edit each note individually.
FindReplaceTools Model Setup
Renumber Elements
Two-step wizard to sequentially renumber rooms, doors, windows, or sheet viewports by clicking elements one at a time. Configure a prefix, start number, and step value before entering the pick loop — press Escape to finish.
RenumberTools Model Setup
Set Metric Units
Sets the project display units to metric: length to mm (0 d.p.), distances to m (3 d.p.), area to m², volume to m³, angle to degrees, and sheet length to mm.
UnitTools Model Setup
Set Imperial Units
Sets the project display units to imperial: length and distance to feet & fractional inches (1/32"), area to SF, volume to CF, angle to degrees, and sheet length to fractional inches.
UnitTools Model Setup
